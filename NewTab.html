Um rotierend verschiedene URLs beim Öffnen eines neuen Tabs zu laden, kannst du jedoch ein kleines HTML-Skript als Umleitung verwenden – genau das, was du brauchst.

Ich zeige dir nun eine angepasste NewTab.html, die bei jedem Öffnen zufällig eine von mehreren URLs auswählt und lädt. Diese Datei kannst du lokal speichern und als Ziel-URL in der Erweiterung angeben (z. B. file:///C:/Pfad/zur/Datei/NewTab.html).

✅ Beispiel: Rotierende Zielseiten

html
<!DOCTYPE html>
<html lang="de">
<head>
  <meta charset="UTF-8">
  <title>Neuer Tab</title>
  <script>
    // Liste der URLs, zwischen denen rotiert wird
    const urls = [
      "https://www.example.com",
      "https://www.wikipedia.org",
      "https://www.heise.de",
      "https://www.spiegel.de"
    ];

    // Zufällige URL auswählen
    const randomUrl = urls[Math.floor(Math.random() * urls.length)];

    // Umleitung ausführen
    window.onload = () => {
      window.location.href = randomUrl;
    };
  </script>
</head>
<body>
  <p>Du wirst weitergeleitet... Falls nicht, <a id="fallbackLink" href="#">hier klicken</a>.</p>
  <script>
    document.getElementById("fallbackLink").href = randomUrl;
  </script>
</body>
</html>

🔧 Was du tun musst:

  1. Ersetze die URLs in der Liste urls = [ ... ] durch deine eigenen Zielseiten.

  3. In der Erweiterung "Custom New Tab URL" gibst du an:

    perl
    file:///C:/Pfad/zur/Datei/NewTab.html

  4. Beim Öffnen eines neuen Tabs wird jetzt zufällig eine URL aus der Liste geladen.
